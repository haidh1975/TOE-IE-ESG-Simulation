## Overview

This repository contains the complete Python simulation code for the paper:

> **Nguyen Tien Minh · Do Huu Hai** (2025).  
> *An integrated TOE-IE-ESG framework linking ESG performance and managerial cognition to digital transformation quality in small and medium-sized enterprises.*  
> *Discover Sustainability*, Springer Nature.  
> Submission ID: `94739b1f-6b60-48f4-880a-088c7e4839d7`  
> Corresponding author ORCID: [0000-0001-5811-7154](https://orcid.org/0000-0001-5811-7154)

### ⚠️ Important Notice

The reference dataset generated by this code is a **calibration-based simulation** anchored to published empirical parameters from prior studies. It is **NOT** independent primary or archival firm-level data. All reported results are **illustrative outputs** of the imposed data-generating structure and should not be interpreted as empirical confirmation of the hypotheses in actual SME populations.

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/[USERNAME]/TOE-IE-ESG-Simulation.git
cd TOE-IE-ESG-Simulation
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv venv
source venv/bin/activate        # macOS / Linux
venv\Scripts\activate           # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the simulation

```bash
# Standard run (N=500, seed=42) — reproduces all paper results
python simulate.py

# Custom seed for reproducibility check
python simulate.py --seed 99

# Larger sample
python simulate.py --n 1000

# Full run including sensitivity analysis (≈3–5 minutes)
python simulate.py --sensitivity

# Specify output directory
python simulate.py --outdir my_results

# All options combined
python simulate.py --n 500 --seed 42 --sensitivity --outdir outputs
```

## Anchor Studies & Calibration Parameters

All data-generating parameters are anchored to the following published studies:

| Source | Parameters Used |
|--------|----------------|
| Ramos-Vecino et al. (2026). *Sustain Technol Entrep*, 5, 100136 | β(IndivF→Dig) ≈ 0.302; β(Dig→Perf) ≈ 0.549; AVE(OpPerf) ≈ 0.477 |
| Guo et al. (2024). China A-share firms OLS | β(ESG→Subst) ≈ 0.11; β(Myopia) ≈ −0.082; β(ESG×Myopia) ≈ −0.058; Kaplan-Zingales FC formula |
| Nykänen et al. (2023). Finnish growth SMEs | ≈ 50% opportunity-dominant cognitive schema |

### Data-Generating Equations (Appendix A.3)

```
SubstDig = 0.110·ESG − 0.082·Myopia + 0.300·IndivF + 0.060·OrgF
           − 0.188·FC − 0.058·(ESG×Myopia) + 0.115·(ESG×IndivF) + ε₁

StratDig = 0.043·ESG + 0.005·Myopia + 0.085·IndivF
           + 0.037·(ESG×Myopia) + ε₂

OpPerf   = 0.549·SubstDig + 0.094·StratDig + 0.104·IndivF + ε₃

FC       = −0.737·ln(Size) + 0.043·ln(Size)² − 0.040·Age    [Kaplan-Zingales]
```

All inputs standardized (mean=0, SD=1) before estimation.  
Error terms: ε₁ ~ N(0, 0.85); ε₂ ~ N(0, 0.92); ε₃ ~ N(0, 0.63)

### Distributional Assumptions (Exogenous Variables)

| Variable | Distribution | Parameters | Bounds |
|----------|-------------|------------|--------|
| ESG | Normal | μ=4.24, σ=0.65 | [1, 9] |
| Myopia | Half-normal | σ=0.091 | [0.01, 0.40] |
| IndivF | Normal | μ=3.09, σ=0.87 | [1, 5] |
| OrgF | Normal | μ=3.20, σ=0.80 | [1, 5] |
| ln(Size) | Normal | μ=3.50, σ=0.80 | — |
| Age | Gamma | shape=2, scale=8 | — |

Exogenous variables generated as approximately independent (|r| < 0.05).

## Sensitivity Analysis

The sensitivity analysis (Appendix A.4) varies all structural coefficients simultaneously by ±20% across 500 Monte Carlo replications and records the proportion of replications in which each hypothesis' coefficient retains its hypothesized direction.

```bash
python simulate.py --sensitivity
```

Expected results (consistent with paper):

| Hypothesis | Reproducible Stability (seed=42) |
|------------|-----------------------------------|
| H1a: β(ESG → SubstDig) > 0 | 99.4% |
| H1b: β(ESG → StratDig) > 0 | 82.0% |
| H3a: β(ESG×Myopia → SubstDig) < 0 | 88.4% |
| H3b: β(ESG×Myopia → StratDig) > 0 | 79.2% |
| H4:  β(ESG×IndivF → SubstDig) > 0 | 98.0% |

**Note:** H3b's 79.2% stability rate is the lowest among all hypotheses, confirming it as the lowest-confidence finding and warranting priority in future primary-data validation. Exact percentages are fully reproducible with `--seed 42`.
